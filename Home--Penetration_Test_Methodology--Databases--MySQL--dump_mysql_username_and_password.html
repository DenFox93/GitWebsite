dump mysql username and password


# MySQL 5.6 and below
mysql> select host, user, password from mysql.user;
# MySQL 5.7 and above
mysql> select host, user, authentication_string from mysql.user;



The password hash stored by MySQL Server is generated by hashing the plain-text password twice using SHA1. When the client transmits the password to the server, it uses three pieces of information:
1. The SHA1 hash of the plain text password
2. The SHA1 hash of the the SHA1 hash of the plain text password
3. The random seed sent by the MySQL Server in the handshake.
