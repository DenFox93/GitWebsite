How works Fragmentation


fragmentation


In order to be sent over the physical network link the data must be encapsulated
In conjunction with the encapsulation, the data has to be fragmented to be small enough to fit the format of the technology being used
To each fragment of the message is assigned an unique identifier called fragmentation ID (IP ID), in this way the receiver knows the correct sequence of the fragments and can reassemble into the original message

Since many operating systems simply increment this number for each packet they send, probing for the fragmentation ID can tell an attacker how many packets have been sent since the last probe.
