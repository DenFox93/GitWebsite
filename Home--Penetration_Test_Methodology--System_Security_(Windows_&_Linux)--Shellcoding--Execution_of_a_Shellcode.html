Execution of a Shellcode


How to proceed:


1. Identify a vulnerable application
2. Inject shellcode in the application
3. adjust EIP(we have to input it in reverse order because written in the Stack) so that it to point to the shellcode

How the Shellcode is executed:


• Adjusting of the EIP and then:
◇ send the shellcode through the network (remote buffer overflows)
◇ set the shellcode through the local environment.
• Overwrite the SEH (Structured Exception Handler) frame
The SEH frames store the address to jump to when there is an exception. If we overwrite the stored address of the SEH frame, the attacker can take control of the execution
