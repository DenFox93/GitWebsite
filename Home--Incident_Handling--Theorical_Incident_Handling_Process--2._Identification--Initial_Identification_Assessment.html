Initial Identification Assessment


Determine whether an event is actually an incident or a false positive
Assess the data to see if it indicates this could be something other than an incident, like simple mistakes by users, admins, or others

Assessment Questions
Complete list of questions by Lenny Zelster: https://zeltser.com/security-incident-questionnaire-cheat-sheet/
If we find a vulnerability ask yourself these questions:
• How widely deployed is the affected platform or application?
• What is the effect of vulnerability exploitation?
• What is the value of the systems impacted so far? What is the value of the data on those systems?
• Can the vulnerability be exploited remotely (via a network connection)?
• ls a public exploit available? Was one recently released?
• What level of skill and prerequisites are required by an attacker to exploit the vulnerability?
• Is the vulnerability present in a default configuration?
• ls a fix available for the vulnerability?
