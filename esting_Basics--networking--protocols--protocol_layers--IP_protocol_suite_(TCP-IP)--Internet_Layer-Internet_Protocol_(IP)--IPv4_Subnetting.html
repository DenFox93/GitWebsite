IPv4 addresses


Any host on a computer network, that can be in a private network or "in the Internet”, is identified by a unique IP address.
Each ipv4 address is formed by 4 bytes(so in total 32 bit):

IP/Mask


To identify a host in a network we need a ip address and a netmask(also called subnetmask)
With an IP/netmask pair, we can identify the network part and the host part of an IP address.
• Network part→ number of bits at the start of the subnet mask set to 1
• Host part→ Number of bits at the end of the subnet mask set to 0
