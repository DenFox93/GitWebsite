msfvenom



In the previous chapter, we used msfvenom to encode a custom payload. Since Metasploit offers a big list of powerful payloads let's see them

Each payload has:
• a target OS
• its own features
◇ meterpreter → interactive connection
◇ cmd → run a single command

List of payloads
msfvenom --list payloads



payload's options
with argument --payload-options
msfvenom -p <name_of_the_msfvenom_payload> --list-options



example: generate a shellcode


msfvenom -p windows/messagebox TEXT="Hello World!" -f c -a x86 --platform windows

-p → sets windows/messagebox as the payload to use
TEXT= → sets “Hello World!” as the text to show in the message box
-f → sets the format of the output to the language code C
--platform → target Windows as platform for the shellcode

To test if the shellcode works properly we can test it wit with the program seen in the chapter “Test if a Shellcode works”
