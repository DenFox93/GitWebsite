Structured Query Language


SQL, pronounced “sequel”, is the most common language(but not the only one) for querying and manipulating data of Relational Databases.

User-Defined Functions
Like functions in programming languages, SQL Server user-defined functions are routines that accept parameters, perform an action, such as a complex calculation, and return the result of that action as a value. The return value can either be a single scalar value or a result set.
These UDFs can be used also to escalate privileges or escape to the underlying OS.
