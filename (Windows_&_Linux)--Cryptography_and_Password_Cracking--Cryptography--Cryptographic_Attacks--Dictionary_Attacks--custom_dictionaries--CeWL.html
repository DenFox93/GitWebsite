CeWL



CeWL written by Robin "DigiNija" Wood, available at https://digi.ninja/projects/cewl.php

scrape a target organization’s website for keywords, and in turn, will generate a list of words we can use for our wordlist.

example:
cewl -d 0 -m 5 -w docswords.txt https://www.offensive-security.com/

-d <number> → by default, sticks to just the site you have specified and will go to a depth of 2 links, this behaviour can be changed by passing a number as argument to -d
-m <number> → minimum word length, default 3.
-w <file> → write the output to the file.
