Data Transfers


PUSH instruction



PUSH stores a value to the top of the stack, causing the stack to be adjusted by -4 bytes (on 32 bit systems): -0x04


PUSH 0x12345678

is the same of:
SUB ESP, 4   ;subtract 4 to ESP -> ESP=ESP-4
MOVE [ESP], 0x12345678   ;store the value 0x12345678 to the location pointed by ESP. 
                         ;Square brackets indicates to address pointed by the register.



POP instruction



POP reads the value from the top of the stack(in case store it in another register), then the stack is adjusted of +4 bytes (on 32 bit systems): +0x04


POP EAX

is the same of:
MOV EAX, [ESP]   ;store the value pointed by ESP(the value at the top of the stack) into EAX
ADD ESP,4        ;Add 4 to ESP in order to adjust the top of the stack
