Defense: how prevent that an attacker cover his tracks
For internet-accessible machines with sensitive data, a great amount of care must be taken with the logs. For internal systems, logging might be less important
• Preparation
◇ Use a separate server for logging (system logs are sent to a remote system)
▪ In UNIX, syslog to a separate server
▪ Windows Event Forwarding (WEF) for Windows systems. It use the built-in WinRM service to pull or push logs to a central repository.
▪ User Behavioral Analytics
▪ Tools to parse and search through logs: Kibana (ELK), Elasticsearch, Logstash
◇ Cryptographic integrity checks of log files
▪ Msyslog from Core Labs includes remote syslog and integrity-checking capabilities
