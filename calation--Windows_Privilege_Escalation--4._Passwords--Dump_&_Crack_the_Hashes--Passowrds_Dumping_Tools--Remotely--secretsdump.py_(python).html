secretsdump.py (python)


Github: https://github.com/SecureAuthCorp/impacket/blob/master/examples/secretsdump.py
It is part of the Impacket Toolkit

To use it we have to type secretsdump.py followed by the target that is in the form:
◇ [[domain/]username[:password]@]<targetName or address>
◇ LOCAL (if you want to parse local files)
secretsdump.py <domain>/<user>:<password>@<IpTarget>

OPTION:
-hashes LMHASH:NTHASH → NTLM hashes, format is LMHASH:NTHASH
