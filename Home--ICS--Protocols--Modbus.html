Modbus


Modbus has come to some revisions during time:
Modbus ASCII
• Serial RS‐232 or RS‐485
Modbus RTU (Most Common)
• Serial RS‐232 or RS‐485
Modbus Plus (Modbus+, MB+)
• Proprietary to Modicon
• Twisted pair up to 1 MB/s
• Uses token rotation
Modbus TCP
• Transported within TCP/IP data packets
• Uses Port 502



Modus message 
Modbus is very easy to use but is not very robust because the packets exchanged are in clear text
Because messages are in clear text (easy to view on a protocol analyzer), are easy to decode, and require no authentication or authorization
