Which method of hash extraction you decide to use depends on the context of your test.
• If we are dealing with an individual system that is NOT a domain controller:
◇ The meterpreter hashdump command
◇ The meterpreter mimikatz(with a meterpreter shell or the standalone .exe) created Benjamin Delpy (@gentilkiwi)
◇ The meterpreter hashdump script (post/windows/gather/hashdump.rb)
◇ The meterpreter smart_hashdump script <-------------- BETTER
• If we are dealing with host based IPS or Anti-Virus that is detecting LSASS injection attempts, we are taking a significant risk when injecting assembly into this process.
In the context of a domain controller, a Volume Shadow Copy Service extraction is the best approach to ensure stability.
