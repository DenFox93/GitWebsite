Control Flow


CALL and RETN instructions


Subroutines (or procedures) are implemented by using the CALL and RETN instruction pair

1. CALL instruction jumps to the function address specified
◇ push the current instruction pointer (EIP) to the stack

2. RETN instruction when the function finish to execute, that automatically jump back to the address directly after the last CALL used
◇ remove the actual EIP from the stack frame and reset it to the location before the time of prologue routine
◇ ESP is automatically updated and point to the nearest value after the CALL
