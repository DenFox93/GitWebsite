IP and MAC adresses, how they work toghether


Every host on a network has both MAC and IP address

How to send a packet from host A to host B?


if host A is in network A and host B in the network B,
and these two networks are connected by a router?

host A have to create a packet with:
• IP header of the datagram:
destination IP address of host B
source IP address of host A
• link layer header of the frame:
destination MAC address of the router
source MAC address of host A

When the router take the packet and have to forward it to host B have to rewrite the packet's MAC addresses without touching IP header:
• NEW link layer header:
destination MAC address of host B
source MAC address of the router
MAC addresses have to be changed so that locally the network knows always where to forward the packet(MAC address next-hop) and where it came from (MAC address last-hop)
