XSS into a template literal with angle brackets, single, double quotes, backslash and backticks Unicode-escaped


XSS:
• into a template literal
• angle brackets Unicode-escaped
• single quotes Unicode-escaped
• double quotes Unicode-escaped
• backslash Unicode-escaped
• backticks Unicode-escaped



1. Submit a random alphanumeric string in the search box (example: daniele93)
/?search=daniele93

use Burp Suite to intercept the search request and send it to Burp Repeater.

Observe that the random string has been reflected inside a JavaScript Template literal (we can recognize them because they are encapsulated in backticks (`) inside a <script>....</script>)
2. Replace your input with the following payload to execute JavaScript inside the template string: ${alert(1)}
/?search=${alert(1)}
