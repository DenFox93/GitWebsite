2. If there are one or more projects in Jenkins, we can check if we have Configure access to one of the them. Click on Configure

3. Now we should be in the folder: “http://<IpJenkinsServer>:8080/job/<PROJECT>/configure”
Here from "Build → Execute Windows Batch Command" we can execute a Powershell command and then click on Save.
If there are in the Project multiple commands, we can insert our command as first in the list, to be sure that it will be executed for first

4. Now we can click on “Build Now”
